Detailed Description

Manages all the memory managers used in a process.

Only once instance of this class exists per process. This instance can be obtained via System::getMasterMemoryManager().

This object is initialized with two memory managers. (1) The manager obtained by getDefaultManager() is the one responsible for ref-counted garbage collection. (2) The manager obtained by getStaticManager() is responsible for objects with unlimited lifetime.
