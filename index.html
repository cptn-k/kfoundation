The KFoundation Story

KFoundation is a made by Kay, a programmer who used to program in C++ a long time ago, but lived most of his life in the harmonious world of Java, and the well-organized Foundation framework of Objective-C. He finds himself again in the world of C++ that even with new revisions as C++11 etc. looks old and backwards, and unnecessarily complicated. BOOST library didn't help because it leaves an extensive footprint on the compiled code. He wanted something small, elegant, yet highly capable, fast and modern. And so, the KFoundation was born.

Features

  • Portable
    • Cross-platform
    • Backwards compatible with any compiler down to C++98
    • Fixed-sized data types
  • Managed Pointers
    • Fastest, safest and most elegant smart pointer implementation to date.
    • Automatic reference counted memory management
    • Supports both heap- and pool-allocated objects
  • System class (as in java.System class)
  • Exceptions with human/machine-readable stack trace
  • I/O Stream
    • Elegant renewed IO Stream API design
    • Extended variations of IO streams: File, Buffer, String, Internet, ...
    • Supplemented with a versatile parser utility
  • Multithreading
  • Managed containers (array) and type wrappers
  • Novel extensible object serialization/deserialization API to/from XML, JSON, etc.
    • Perfect for RESTful applications
    • Valuable tool for debugging your objects
  • Region arithmetics
  • Multi-channel Multi-level Logger
    • LOG << "Logging had never been so easy!!!" << EL

Want to Get It?

The best way to get KFoundation is by cloning its git repository to your local machine.

$git clone https:://github.com/hkhandan/kfoundation.git

Alternatively, you can download the latest release from https://github.com/hkhandan/kfoundation/releases page.

To install you need

  • CMake 3.0 or above
  • A Compatible C++98 or above compiler (gcc or clang would do)
$cd kfoundation
$mkdir build
$cd build
$cmake ..
$make
$sudo make install
